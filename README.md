# 🚗 Car Price Predictor

This is my first Machine Learning project where I built a model to predict car prices based on different features.

The main goal of this project was to understand how feature selection, model training, and deployment work in a real-world scenario.

## 📌 About the Project

In the beginning, the model was giving average results.  
I kept improving it by selecting better features and testing different combinations.

After multiple iterations, I achieved a stable and better-performing model and deployed it using Streamlit

## ⚙️ Features Used

- Engine Size  
- Horsepower  
- City MPG  
- Curb Weight  
- Car Width  
- Fuel Type (Gas/Diesel)  
- Drive Wheel (FWD / RWD / 4WD)

## 📊 Model Performance

- R² Score: 0.827  
- Mean Squared Error: 13642072  
- Mean Absolute Error: 2474  
- Training R²: 0.832  
- Adjusted R²: 0.784
